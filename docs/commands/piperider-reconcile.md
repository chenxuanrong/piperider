
# piperider reconcile
Compare dataset defined in reconcile.yml.
## Usage
```
Usage: piperider reconcile [OPTIONS]
```
## Options
* `report_dir`: 
  * Type: STRING 
  * Default: `none`
  * Usage: `--report-dir`

  Use a different report directory.


* `output`: 
  * Type: STRING 
  * Default: `none`
  * Usage: `--output`

  Director to save the results


* `project`: 
  * Type: STRING 
  * Default: `none`
  * Usage: `--project`

  Project name


* `debug`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--debug`

  Enable debug mode.


* `help`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
